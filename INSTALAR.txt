===================================================================================
+ Instalación
===================================================================================

1º  Dependencias : Instalar estos paquetes de los repositorios de:
		GNOME:
			sudo apt-get install imagemagick wget rar libssl-dev
		
		KDE:
			sudo apt-get install imagemagick wget rar libssl-dev python-gtk2 python-glade2 gnome-icon-theme menu
			
2º  Instalar :
		sudo make install
